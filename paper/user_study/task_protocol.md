# 用户测试任务清单（线上电子版 v2）

> 2026-04-18 重写：面向 10 名非生信研究生线上测试（腾讯问卷电子版）。
> 原版 10 任务改 **8 任务 / 30-40 分钟**，强调 UI 交互 / 功能齐全 / 新手引导
> 三维度验证。

## 总体约定

- **时间预估**：30-40 min（含读问卷 / 操作 / 截图 / 填问卷）
- **环境要求**：电脑浏览器（Chrome / Edge / Firefox 任一）；需联网访问
  EnvMeta 部署地址
- **数据来源**：EnvMeta 内置「📦 加载砷渣修复示例数据」按钮一键加载，
  **不需要你准备任何文件**
- **遇到问题**：跳过当前任务，在问卷对应题目"备注"栏写下卡点描述
  + 截图，继续做下一题。**所有任务允许跳过**
- **截图要求**：Windows 用 `Win+Shift+S` / Mac 用 `Cmd+Shift+4`；
  **上传到问卷里**（腾讯问卷支持图片上传）
